brAInstormer is a fully-featured, browser-based creative suite. Chat, generate images, edit audio, cut video, view 3D models, write code — all in one dark-themed app, all supercharged by the best AI models. Connect your own API keys and keep everything local.

> **Zero data sent to our servers.** All AI calls go directly from your browser to the AI provider using your own keys.

## ✨ Modules

| Module | What it does |
|---|---|
| **AI Chat** | Multi-model chat — GPT-4o, Claude Sonnet, Llama 3. Text, code, images in one thread |
| **Image Editor** | Layer-based editor with AI generation (DALL·E 3, Stable Diffusion, Flux) + inpainting |
| **Audio DAW** | Multi-track recording, sequencing, mixing. MusicGen + Stable Audio. Full piano roll |
| **Video Editor** | Non-linear timeline, multi-track, AI-assisted cuts |
| **3D Viewer** | Import and inspect 3D models in the browser |
| **Code Editor** | Syntax highlighting + AI pair programmer built in |

## 🚀 Quick Start

```bash
git clone https://github.com/RhythrosaLabs/brainstormer-4.git
cd brainstormer-4
npm install
npm run dev
```

Open `http://localhost:5173`, add your API keys in Settings, and start creating.

## 🛠️ Tech Stack

- **React 18 + TypeScript** — component architecture
- **Vite 5** — blazing-fast dev server and build
- **Tailwind CSS** — styling
- **OpenAI** — GPT-4o, DALL·E 3
- **Anthropic** — Claude Sonnet
- **Replicate / Stable Diffusion** — image and audio generation
